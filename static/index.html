Docs

this is the api documentation to help you get started with using it
GET method:
this is used for retreiving info from the api
/alumni/:criteria
/lowongan/:criteria
/jurusan/:criteria

:criteria = search criteria ('all' if you want to list everything or you can put the id to search for specific id)


POST method:
this method is used for adding data to the api
/alumni
-> json body -> nama, tahunlulus, jurusan, status
/lowongan
-> json body -> judul, deskripsi, kemampuan, jenis


PATCH method
this method is used for updating data in the api
/alumni
/lowongan
-> json body -> where, fix, id -> where (what you want to change i.e nama tahunlulus status), fix (what do you want to change it to), id (the id of the data you want to change)


DELETE method
/alumni/:id
/lowongan/:id
-> id -> specify which one you want to change
