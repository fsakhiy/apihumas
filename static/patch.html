PATCH /update/:column

digunakan untuk mengupdate data

  • Parameter

    • parameter :column ini diisi kolom yg mau diganti

  • Req Body

      • digunakan untuk memilih data yg diganti (alumni, lowongan)

    • changes

      • ini diisi dengan gantinya data
