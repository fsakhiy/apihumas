Authentication

  • GET
  • POST
  • PATCH
  • DELETE
  • AUTHENTICATION

untuk kebanyakan method anda perlu melakukan authentikasi

  • signup

    • kamu akan memerlukan signup untuk membuat user baru
    POST /signup
    req body = json = username, password, idAlumni, admin, email

  • login

    • login akan dibutuhkan untuk membuat access token dan refresh token
    POST /login
    req body = json = username, password

  • autentikasi

    • ini akan kamu butuhkan untuk mengakses api itu sendir
    POST or GET or PATCH or DELETE /(yg kamu akan akses)
    di req body = json = ditambahkan authtoken ({ "authtoken": "(access token dari POST /login)"})

  • logout

    • ini digunakan ketika user mau logout dan berfungsi untuk membuat refreshtoken expired
    DELETE /logout
    di req body = json = refreshToken ({"refreshToken": "(refreshtoken dari post /login)"})

  • reset password

    • ini digunakan untuk ketika user lupa password.
    UNTUK RESET PASSWORD AKAN DILAKUIKAN DILUAR APLIKASI, APLIKASI BERFUNGSI HANYA UNTUK MENGIRIM EMAIL RESET PASSWORD
    untuk merequest email untuk reset password
    di req body = json = email ({"email": (email user)})
    sisanya akan dihandle external
